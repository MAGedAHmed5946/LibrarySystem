# Main Library System

A professional Library Management System built with Python, Tkinter, and an SQLite database.
Refactored into an object-oriented format designed to allow for multiple team members merging and working together seamlessly on Github.

## Project Structure
- `main.py` - Application entry point logic and routing architecture.
- `database.py` - Core SQLite DB setup.
- `gui/` - Contains logic specific to each page display mechanism.
- `assets/` - Image repository for app backgrounds and icons (Reserved/Not currently in usage)

## Setup
1. Clone the repository structure
2. Verify you have Python installed. You do not need an environment as Tkinter + Sqlite are in the standard Python Library 
3. Run `python main.py` at the root folder path.
